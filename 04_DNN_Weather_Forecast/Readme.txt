Project Title:
---------------
Deep Neural Networks modeling for Weather Forecasting.

Objective:
-----------
Predict Mean Sea Level Pressure, Minimum and Maximum Temperature and Dew Point Temperature weather charateristic
using Hong-Kong Observatory time-series dataset.

Model Developed:
----------------
Stacked Auto-Encoder DNN, Non-Linear AR Model (for comparison)

Tools used:
-----------
Matlab Neural Networks toolbox
